# COGOS (Swift)

Pure-Swift / SwiftUI port of the COGOS app. iOS-only. Targets iOS 14+.

## Xcode project setup

A `COGOS.xcodeproj` is not committed — create it fresh in Xcode:

1. **File → New → Project → iOS → App**.
2. Product Name: `COGOS`. Interface: **SwiftUI**. Language: **Swift**.
   Bundle ID: `com.jackhu.cogos`. iOS Deployment Target: **14.0**.
3. Save inside the repo root (alongside `COGOS/`). Delete the auto-generated
   `ContentView.swift` and `COGOSApp.swift` placeholders.
4. Drag the `COGOS/` directory into the project navigator. Choose
   **Create groups** and **Add to target: COGOS**. Ensure all `.swift`, `.m`,
   `.c`, `.h` files get added.
5. In target settings:
   - **Build Settings → Objective-C Bridging Header**:
     `COGOS/Supporting/COGOS-Bridging-Header.h`
   - **Info → Custom iOS Target Properties**: replace the autogenerated
     `Info.plist` with `COGOS/Supporting/Info.plist`.
   - **Signing & Capabilities**: add **Background Modes → Uses Bluetooth LE
     accessories**.
6. Build & run on a physical device (BLE can't be simulated).

## API keys

Either enter them in the in-app Settings screen (they persist to
`UserDefaults`), or export `ANTHROPIC_API_KEY` as an environment variable in
the Xcode scheme.

